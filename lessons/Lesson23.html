Lesson 23: Input

But what if you want to get the computer to ask whomever that is using it for a name?

To do that, use the await input statement.

The code below will let the computer ask the user for name, and store it in a variable called name.

var name = await input ("What is your name:")

After getting the user's name we can print our a customized greeting message using the user's name.

print ("Hello " + name)
