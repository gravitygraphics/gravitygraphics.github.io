Lesson 19: For Loop

To plot an extra dot, we just need to add in an extra line of code. But what if we want to plot 200 pixels? It would be a lot of work to add in 200 lines of code.

Here's where the for loop can make our work a lot easier.

for (count=0; count < 200; count=count+1) {
    dot(random(0,59), random(0,39), random(0,15))
}

We just put our code into the for loop and have the for loop repeat our code 200 times.
